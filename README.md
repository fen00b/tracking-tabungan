# 💰 Dompetku — Personal Money Tracker

Aplikasi money tracker pribadi yang bisa diakses dari browser, tanpa perlu install apapun.

## Fitur
- Catat pengeluaran & pemasukan tambahan (uang makan, bonus, dll)
- Pilih tanggal transaksi — bisa input data masa lalu
- Alokasi gaji otomatis (kebutuhan harian, sewa, BPJS, listrik, internet, tabungan)
- Hitung sisa budget harian & per hari hingga gajian berikutnya
- Data tersimpan di browser (localStorage)
- Export ke CSV
- Tampilan mobile-friendly

## Deploy ke GitHub Pages

1. Buat repository baru di GitHub (contoh: `dompetku`)
2. Upload file `index.html` ke repository tersebut
3. Buka **Settings → Pages**
4. Pilih **Source: Deploy from a branch → main → / (root)**
5. Klik Save
6. Tunggu beberapa menit, lalu akses di: `https://username.github.io/dompetku`

## Catatan
- Data disimpan di localStorage browser — tidak sinkron antar device
- Untuk backup, gunakan fitur Export CSV
